# Palindrome Number – Resolução em Java

## Descrição do Problema

  O método isPalindrome receberá um número inteiro "x", e irá retornar "true" se ele for um palíndromo, e "false" caso contrário.

  Um número é considerado palíndromo quando ele é lido da mesma forma da esquerda para a direita e da direita para a esquerda.

## Exemplos

### Exemplo 1

  Entrada: x = 121<br>
  Saída: true<br>
  121 → lido da esquerda para a direita<br>
  121 → lido da direita para a esquerda<br>

### Exemplo 2

  Entrada: x = -121<br>
  Saída: false<br>
  -121 → esquerda para direita<br>
  121- → direita para esquerda<br>

### Exemplo 3

  Entrada: x = 10<br>
  Saída: false<br>
  10 → esquerda para direita<br>
  01 → direita para esquerda<br>
