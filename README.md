# Palindrome Number – Resolução em Java

## Descrição do Problema

  O método isPalindrome receberá um número inteiro "x", e irá retornar "true" se ele for um palíndromo, e "false" caso contrário.

  Um número é considerado palíndromo quando ele é lido da mesma forma da esquerda para a direita e da direita para a esquerda.

## Exemplos

### Exemplo 1

  Entrada: x = 121
  Saída: true
  121 → lido da esquerda para a direita
  121 → lido da direita para a esquerda

### Exemplo 2

  Entrada: x = -121
  Saída: false
  -121 → esquerda para direita
  121- → direita para esquerda

### Exemplo 3

  Entrada: x = 10
  Saída: false
  10 → esquerda para direita
  01 → direita para esquerda
